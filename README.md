

# RankLens — Keyword Rank Checker

A lightweight, cost-optimised **Google & Bing SERP rank tracking tool** built as a single-file HTML app, powered by the [DataForSEO API](https://dataforseo.com/).

## ✨ Features

- 🔍 **Bulk keyword rank checking** — up to 1,000 keywords per run
- 🌍 **Granular geo-targeting** — countries, states, and cities across India, USA, UK, Australia, and more
- 📊 **Live progress tracking** — results stream in as checks complete
- 🏷️ **Rank badge color-coding** — Top 3 / Top 10 / Top 30 / Not Found at a glance
- 🔎 **SERP viewer** — inspect full result pages per keyword
- 📥 **CSV export** — download all results with one click
- 💰 **~40% cheaper** than full enrichment endpoints — no search volume padding, same accuracy
- 🖥️ **Desktop & Mobile** device targeting
- **Google & Bing** search engine support
- Zero backend required — runs entirely in the browser

## 🚀 Usage

1. Open `ranklens.html` in any modern browser
2. Enter your **DataForSEO API login & password**
3. Set your **target domain**, **location**, and **keywords** (one per line)
4. Click **Check Rankings**

> Credentials are stored only in your browser session and never sent anywhere except the DataForSEO API.
